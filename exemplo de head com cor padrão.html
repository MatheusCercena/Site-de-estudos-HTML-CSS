Veja um exemplo de head com cor padrão

        
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Conteúdo de HTML</title>
    <style id="cor-padrão-de-marcação">
        mark {
            background-color: lime;
        }
    </style>
    <!--usar style dentro de head para adicionar uma característica visual padrão à uma tag-->
    <!--nesse caso, é uma regra geral, mas pra fugir da regra, pode-se usar style dentro da tag que se quer mecher-->
    <link rel="shortcut icon" href="imagens/Paisagem_da_Caatinga_Paraibana-1280.JPG" type="image/x-icon">
</head>
