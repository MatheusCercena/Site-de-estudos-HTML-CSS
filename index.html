5.3 Usando <picture> + <img>

Permite o uso de imagens de diversas resoluções diferentes, para tornar o site otimizado e dinâmico.

Usa-se a tag <pic> para envelopar as tags <img> e <source media> em que:

picture
indica que o que estará envelopado é uma imagem.
img src
É a imagem original a ser mostrada para sites em janelas grandes.
source media
Deve vir acima de "img src" e conter a próxima menor imagem, diminuindo gradativamente.

                    
<source media="(max-width: 1230px)" srcset="imagens/Paisagem_da_Caatinga_Paraibana-640.JPG" type="image/JPG">
<source media="(max-width: 1919px)" srcset="imagens/Paisagem_da_Caatinga_Paraibana-1280.JPG" type="image/JPG">
<img src="imagens/Paisagem_da_Caatinga_Paraibana-1920.JPG" alt="Imagem Flexível">

Deve-se remover 1 pixel em "max-width", de forma que se a tela é 1920p, assim que o tamanho dela for diminuído irá carregar a imagem imediatamente menor.

Tente abrir essa imagem em janelas de tamanhos diferentes no navegador.
Dica: o atalho do windows+shift+m aciona uma ferramenta do App Power Toys chamada "Screen Ruller", que permite medir o tamanho da tela, o que é util para testar o tamanho de tela em navegadores e ver como as imagens correspondem.

16.5 Outros atributos importantes

  • required: dentro de input, para indicar que o formulário é de preenchimento obrigatorio
  • minlength e maxlength: quantidade minima e maxima de caracteres
  • placeholder: texto que fica levemente apagado dentro do formulario
  • input type="reset": limpa o conteúdo escrito no formulario
  • autocomplete: off : dentro da tag form, para desligar o autocomplete do navegador
  • autocomplete: username and current-password: dentro da tag input para ligar o preenchimento automatico do navegador e relacionar com o gerenciador de senhas
  • input type=
  • min e max = quantidade minima e maxima de caracteres numericos
  • value = dentro da tag input substitui o placeholder e coloca um valor dentro ca caixa de formulário
  • step, dentro de input type number para indicar de quando em quanto os numeros pulam
  • input type="date", "month", "time" - para colocar datas (month não é compatível com Firefox)
  • O atributo email - usar com autocomplete="email"
  • O atributo tel pode ser usado co autocomplete="tel", ele não permite filtros, mas pode-se usar expressões regulares, chamadas de RegEx(estudar sobre) com o atributo pattern dentro da tag input para criar filtros manuais

17.0 Media Queries

17.1 O que são

São estilos em css feitos para que o site possa ser visualizado em diferentes tipos de telas. Como telas em retrato, paisagem, impressoras, etc. Cria-se 1 ccs para cada, sendo usado 1 css com o atributo media="all" com configurações globais e arquivos separados com media="screen", ou media="print" ou media="screen and (orientation: landscape)" ou media="screen and (orientation: portrait)"

Clique aqui para ir a pagina do site em que pode ver media queries separadas em ação para resposividade

Clique aqui para ir a pagina do site em que pode ver media queries unificadas em ação para resposividade

17.2 Device breakpoints

Dentro de uma media query, se usa a identificação @media screen (min-width: xxxx) {...} para definir uma media query para cada tamanho de tela. Veja nesse código:

                            
        /* Telas pequenas */
        @media only screen and (max-width: 599px) {...}

        /* Celulares pequenos e navegadores em tela dividida*/
        @media only screen and (min-width: 600px) {...}
        
        /* Celulares normais em retrato e telas antigas 1366x768p */
        @media only screen and (min-width: 1080px) {...}
        
        /* Celulares em paisagem e telas FullHD */
        @media only screen and (min-width: 1920) {...}
        
        /* TVs e monitores 4k e 1080p */
        @media only screen and (min-width: 2560px) {...}
